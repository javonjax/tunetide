# Tunetide - Radio Browser

## Description

Tunetide is a full-stack web application that allows users to explore and listen to radio stations from around the world in real time. The app provides an intuitive interface for discovering global radio content across different countries and genres.

## Technologies Used

- [Next.js](https://nextjs.org/)

- [SupaBase](https://supabase.com/)

- [ShadnCN UI](https://ui.shadcn.com/)

- [TailwindCSS](https://tailwindcss.com/)

- [OAuth](https://oauth.net/2/)

The application is deployed using Vercel.

- [Vercel](https://vercel.com)

## Features
- Access to countless radio stations from around the world.
- Find radio stations by popularity, recent activity, genre, location, and more.
- Create an account to save your favorite stations.
- Trending stations tag and recent listener activity visualization.

## Getting Started

First, clone this repo to your machine.

Then install dependencies:
```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
